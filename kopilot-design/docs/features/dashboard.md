# 대시보드 페이지

## 페이지 개요

로그인 이후 가장 먼저 등장하는 메인 페이지이다.

마이데이터를 기반으로 사용자의 소비 데이터를 분석하고, 사용자가 자신의 소비 현황을 한눈에 확인할 수 있도록 구성한다.

---

## 주요 사용 데이터

### `merchantName`

사용자가 결제한 가맹점명이다.

예시:

```text
스타벅스 부산대점
배달의민족
카카오택시
```

### `approved_dtime`

사용자가 결제한 날짜와 시간이다.

예시:

```text
2026-07-12 13:25
```

### `approved_amt`

가맹점에서 사용한 결제 금액이다.

예시:

```text
15,000원
```

---

## 화면 구성

### 1. 상단 영역

* 사용자 이름과 환영 문구
* 조회 기간 선택 버튼

예시 문구:

```text
진원님, 이번 달 소비를 확인해 보세요.
```

조회 기간:

```text
이번 달
최근 3개월
최근 6개월
```

---

### 2. 소비 요약 카드

이번 달 소비 현황을 요약해서 보여준다.

표시 정보:

* 이번 달 총소비 금액
* 지난달 대비 증감률
* 총결제 건수

예시:

```text
이번 달 소비

1,245,000원
지난달보다 8.3% 증가
총 42건
```

총소비 금액을 가장 크게 강조한다.

---

### 3. 소비 추이 차트

기간에 따른 소비 금액 변화를 차트로 보여준다.

* 일별 또는 월별 소비 금액 표시
* 선 그래프 또는 막대그래프 사용
* 마우스를 올리면 날짜와 금액 표시

예시:

```text
7월 12일
82,500원
```

---

### 4. 카테고리별 소비

소비 데이터를 카테고리별로 나누어 보여준다.

카테고리 예시:

* 식비
* 카페
* 교통
* 쇼핑
* 문화·여가
* 생활
* 기타

표시 정보:

* 카테고리명
* 카테고리별 소비 금액
* 전체 소비에서 차지하는 비율

도넛 차트 또는 가로 막대그래프 형태로 표현한다.

예시:

```text
식비
420,000원
전체 소비의 33.7%
```

---

### 5. 소비 인사이트

사용자의 소비 데이터를 분석한 결과를 짧은 문장으로 보여준다.

카드 형태로 최대 3개까지 표시한다.

예시:

```text
이번 달에는 식비에 가장 많은 돈을 사용했어요.
```

```text
지난달보다 카페 소비가 22% 증가했어요.
```

```text
주말 소비가 평일보다 높은 편이에요.
```

아이콘이나 간단한 일러스트를 함께 사용할 수 있다.

---

### 6. 최근 결제 내역

최근 결제 내역을 목록 형태로 보여준다.

각 결제 내역에 표시할 정보:

* 가맹점명
* 소비 카테고리
* 결제 날짜와 시간
* 결제 금액

예시:

```text
스타벅스 부산대점
카페 · 오늘 오후 1:25
-5,500원
```

최근 결제 내역은 기본적으로 5개까지 표시한다.

하단에 `전체 내역 보기` 버튼을 배치한다.

---

### 7. 자주 이용한 가맹점

사용자가 자주 결제한 가맹점을 카드 또는 목록 형태로 보여준다.

표시 정보:

* 가맹점 로고 또는 아이콘
* 가맹점명
* 이용 횟수
* 총사용 금액

예시:

```text
스타벅스
8회 이용
총 42,500원
```

## 레이아웃

데스크톱에서는 다음과 같이 구성한다.

```text
상단 사용자 정보 및 기간 필터

소비 요약 카드

소비 추이 차트        카테고리별 소비

소비 인사이트

최근 결제 내역        자주 이용한 가맹점
```

모바일에서는 모든 영역을 세로 1열로 배치한다.

---

## 디자인 방향

* 금융 서비스의 신뢰감을 유지하면서 친근한 분위기로 구성한다.
* 복잡한 금융 용어보다 이해하기 쉬운 표현을 사용한다.
* 금액과 핵심 수치는 크게 강조한다.
* 카드 기반 UI를 사용한다.
* 카드 모서리는 둥글게 표현한다.
* 카드 사이에는 충분한 여백을 둔다.
* 차트와 숫자를 활용하되 화면이 복잡해 보이지 않도록 한다.
* 소비 증가와 감소 상태를 색상과 아이콘으로 구분한다.
* 데스크톱과 모바일에 모두 대응하는 반응형 디자인으로 구성한다.
